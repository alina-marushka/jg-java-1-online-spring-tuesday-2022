Какие значения принимают свойства класса по умолчанию
(ссылки на объекты и примитивные типы данных)
при вызове дефолтного конструктора (конструктора по умолчанию)?

Создайте класс Customer:

class Customer {

    private String nickname;
    private int messagesCount;
    private boolean isOnline;
    private Boolean isModerator;

    // создайте getter методы для каждого свойства

}

Создайте класс CustomerDemo, в которой
внутри main() метода продемонстрируйте ответ на поставленный
в начале задания вопрос.

Для этого просто распечатайте на экран
значения каждого свойства в отдельной строке. Для получения
значения свойства воспользуйтесь соответствующим getter методом.
